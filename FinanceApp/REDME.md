# Finance Tracker App

This is a simple financial tracking application made using **TypeScript**, **HTML**, and **CSS**.

## What It Does

- You can add **income** with a source, amount, and date.
- You can add **expenses** with a source, amount, and date.
- You can add **saving goals** with a goal name and amount.
- All data is saved in the browser using **localStorage**.
- The app works on a single page.
